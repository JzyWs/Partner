一、缩进
使用4个空格作为缩进

二、变量命名
变量名使用拼音或者英语，按照其具体含义进行命名，有的临时变量使用count1~count7命名
例如：
	wuhan：武汉
        nowTime：现在的时间
        wuhancontact：武汉接触史
 	Name_i：获取数据库中的第i行的姓名
 	count1：武汉籍的人数
	count2：湖北籍的人数（非武汉）

三、函数、类命名
函数和类的命名取决于函数或者类的功能
例如：
	BaseDao：数据库基础操作类
        isEffectiveDate（）：判断时间是否符合约束条件
	YiqService():读取数据库的服务类
	FromDbToExcel()：导出Excel、可视化类

四、空行规则
在作用相同但是变量名不同的语句间加入空行
例如：     //从系统获取当前时间
	SimpleDateFormat df = new SimpleDateFormat("yyyy-MM-dd HH:mm");
    	String now =df.format(new Date());	
    	
    	//从系统获取是哪一天
    	SimpleDateFormat df2 = new SimpleDateFormat("yyyy-MM-dd ");
    	String start =df2.format(new Date());
    	
    	//从系统获取是哪一天
    	SimpleDateFormat df3 = new SimpleDateFormat("yyyy-MM-dd ");
    	String end =df3.format(new Date());

五、注释规则
在类或者函数前用/***内容*/进行注释，方法内部单行注释，在被注释语句上方另起一行，使用//注释。方法内部多行注释使用/ *内容*/注释，注意与代码对齐
例如：类或函数：/**
                           * 数据库操作基础类
                           *利用泛型和反射机制来抽象数据库基本的增删该查操作
                           */
 方法内部和单行注释：/*判断时间是否在每天十点以前*/
